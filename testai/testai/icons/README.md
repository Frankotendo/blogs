# Icons Directory for UniHub Live Tracking

Icon files in this directory:

1. **car.svg** (32x32 pixels) - Blue driver vehicle marker
2. **passenger.svg** (32x32 pixels) - Green passenger location marker

These SVG icons are automatically loaded by the tracking.js module for map markers.

## Icon Design:
- **Car**: Blue background with white vehicle silhouette
- **Passenger**: Green background with white person silhouette

Both icons are 32x32 pixels with rounded corners for modern appearance.
